# StudentManagement
基于 Maven 管理的Spring + SpringMvc + MyBatis搭建的学生信息管理系统，包括了学生信息管理和学生课程信息管理。此项目作为SSM入门基础整合项目。

## 技术栈：
 - 后台：Spring、SpringMVC、MyBatis以及Pagehelper、JSR303的使用
 - 前台：Bootstrap
 - 项目管理：Maven、Git
